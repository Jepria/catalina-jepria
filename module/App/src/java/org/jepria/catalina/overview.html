Описание

Данный модуль содержит функционал, встраиваемый в сервер Tomcat, для реализации технологии Sso на кластере с инстансами, установленными на разных физических компьютерах (в частности, кастомизированный realm JepJAASRealm). Для обеспечения максимальной лёгкости и автономности, функционал запечатывается в jar-файл с минимальным количеством зависимостей.

Подготовка бинарного репозитория

Для возможности установки данного модуля необходимо сперва развернуть бинарный репозиторий на локальном окружении.
Для этого достаточно скачать модуль Bin в подходящую директорию на локальной файловой системе ОС, ссылка на которую будет использована во время установки модуля.

Встраивание в Tomcat

Встравиание компонента в Tomcat производится копированием в папку lib библиотеки catalina-jepria.jar с последующей перезагрузкой сервера. Кастомизированный realm используется всеми приложениями на данном сервере, META-INF/context.xml которых содержит явное указание 
<Realm className="org.jepria.catalina.realm.JepJAASRealm"/>
